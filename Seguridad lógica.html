Seguridad Lógica

  • Si bien es cierto la seguridad física es muy importante, nuestra empresa, entidad, organización debe ser más precavida y con más sistemas de seguridad en la parte lógica, el resguardar la información es el deber y compromiso más importante por lo tanto se debe contar con técnicas que blinden los datos y evitar fugas de información. Los datos y la información únicamente pueden tener acceso los usuarios autorizados ya sea para reescribir, ejecutar o únicamente para enviar información, se debe contar con supervisión y alerta ante cualquier riesgo o modificación de permisos a usuarios para realizar o ejecutar acciones no autorizadas Controles de acceso: Es de mucha ayuda para proteger los datos y la información ya que restringe la modificación no autorizada, con lo cual se conserva la integridad de los datos (filtro de usuarios y acciones permitidas) para resguardar información confidencial de usuarios con restricciones.
